First Steps

This guide will help you integrate Flyway into your project and configure it to fit your needs.

Sample application

First, have a look at the sample application available on the Downloads tab.
This shows you the simplest setup possible for Flyway. It'll give you a quick impression of how Flyway can be used and what both Sql and Java migrations look like.
It contains a sample configuration for the Maven Plugin and the Ant Tasks.

Database setup

Now it's your turn. Start by preparing your database.

Which situation applies to you?

My database is empty

Great. Just point Flyway at your database and it'll take care of the rest. No special preparation is required.

I want to start using Flyway with an existing database

Perfect. We have a guide to lead you through the different steps.

Maven plugin setup

When your database is set up, you can give the Maven Plugin a go.

Try 

mvn flyway:status

and check that Flyway can connect to your database.

Ant tasks setup

Alternatively you can give the Ant Tasks a go.

Execute a target containing 

<flyway:status />

and check that Flyway can connect to your database.

Application integration

After you've verified that the Maven plugin is working, the next step is to integrate Flyway in your application.

Write your first migration

Congratulations, Flyway is now fully set up. Manually updating databases on different machines is now officially a thing of the past.

Start by writing your first migration and you'll never look back. :-)
