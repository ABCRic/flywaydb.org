Ant Task: Clean

Drops all objects in the schema without dropping the schema itself.

Usage

<flyway:clean />


Configuration

Attribute Ant / System Property Required Default Value Description
driver flyway.driver YES The fully qualified classname of the jdbc driver to use to connect to the database
url flyway.url YES The jdbc url to use to connect to the database
user flyway.user YES The user to use to connect to the database
password flyway.password NO blank The password to use to connect to the database
schemas flyway.schemas NO default schema of the connection Comma-separated list of schemas managed by Flyway. The first schema in the list will be the one containing the metadata table
classpath flyway.classpath NO The Ant classpath used to load the JDBC driver and the migrations
classpathref flyway.classpathref NO The Ant classpath reference used to load the JDBC driver and the migrations


Sample configuration

<property name="flyway.driver" value="org.hsqldb.jdbcDriver"/>
<property name="flyway.url" value="jdbc:hsqldb:file:/db/flyway_sample"/>
<property name="flyway.user" value="SA"/>
<property name="flyway.password" value="mySecretPwd"/>
<property name="flyway.schemas" value="schema1,schema2,schema3"/>
