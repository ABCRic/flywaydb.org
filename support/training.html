Training

We offer high quality professional training courses to quickly get you up to speed and teach you all there is to know about effective database migrations with Flyway.

Effective database migrations with Flyway

This “in-depth” intensive course equips your development team with advanced know-how of Flyway-based database migration - all through small, informative hands-on modules.

Overview

Join us for one day of intensive Flyway instruction. This class begins with a quick overview of what problems database migrations tools set out to solve. It then quickly moves into thorough coverage of everything from Flyway basics to topics like project integration, API and build tool integration, test data management and backward compatible migrations.

You'll spend around half of your time actually hands-on with Flyway, and the other half talking with the instructor and learning directly from his live coding exercises. This class can take you from complete newcomer status to Flyway competence, or from passing familiarity to Flyway and database migrations expertise. Don't miss it.

Prerequisites

This courses assumes a good understanding of the Java language, SQL and relational databases in general.

Program

During the course of this day, we will cover the following topics:

  • Intoduction to database migrations
  • Installing Flyway
  • Flyway's core functionality and API
  • Ant, Maven and Command-line integration
  • Effective integration into new and existing projects
  • Advanced Flyway features
  • Test data management
  • Backward compatible migrations
