Supported Databases

Database Version
Oracle 10g and later
SQL Server 2008 and later
DB2 9.7 and later
MySQL 5.1 and later
Google Cloud SQL 1.6 and later
PostgreSQL 9.0 and later
H2 1.2.137 and later
Hsql 1.8 and later
Derby 10.8.2.2 and later

Older versions may work, but they have not been tested.

If your database is not on this list, please raise an issue and we'll see what we can do to support it.
