Roadmap

The general priority is first bug fixes and documentation improvements. Then come new features.

Flyway 1.8

  • Issue 99 "Repair" metadata table after failed migration
  • Issue 138 Support applying migrations with older version numbers that haven't been applied yet
  • Issue 199 Trimming of whitespaces in SQL migrations breaks stored procedures formatting.
  • Issue 248 New Flyway.info() call for info about both applied and pending migrations and current status of the DB
  • Make commons-logging dependency optional

  • Issue 74 Support Database/Schema Creation

  • Issue 223 Customizible location of jars folder
  • Issue 277 Allow to load sql-migrations from different directory

  • Issue 274 Batch updates of SQL patches
